Dados Absolutos Relativos

Dados absolutos coleta direta da fonta sem nehum tipo de manipulação

Dados Relativos dados de facil entendimento gerado atraves de dados absolutos e que ajuda a tirar conclusões.

Operações que se podem ser efectuadas com os Dados Relativos

  • Percentagem
  • Indices
  • Coeficientes
  • Taxas

Percentagem

A ideia de percentagem e destacar a participacao da parte no todo para que se possa fazer comparativos.

Indices

O intuito é trabalhar razoes entre duas grandezas,consiste em atribuir um só numero o comportamento geral de uma variavel

Coeficientes

Razoes entre o numero de ocorrencia e o numero total

Formula para caclular o Coeficientes

(Primeira ocorrencia - Ultima ocorrencia) / Primeira ocorrencia
